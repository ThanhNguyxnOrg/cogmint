### Added
- Initial public release of COGMINT
- Dark-first cobalt/violet design system
- Agent orchestration workspace (metadata, prompts, memory)
- Chat and Studio iteration loops
- Workflow builder and relationship graph
- Skills, commands, plugins, and MCP management surfaces
- English-first i18n foundation with Vietnamese translation coverage
- Claude Code plugin entry (`/cogmint`) and OS-specific launcher installers
- GitHub Releases workflow for cross-platform packaging
